> Add gif avatars

## Install

```sh
composer require nearata/flarum-ext-gif-avatars:"*"
```

## Update

```sh
composer update nearata/flarum-ext-gif-avatars:"*"
php flarum cache:clear
```

## Remove

```sh
composer remove nearata/flarum-ext-gif-avatars
php flarum cache:clear
```
